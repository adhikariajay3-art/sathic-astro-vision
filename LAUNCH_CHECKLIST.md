## Customer Flow

1. Customer opens live website.
2. Customer clicks Digital Product Store.
3. Customer selects product.
4. Customer pays by UPI.
5. Customer sends order and screenshot by WhatsApp.
6. Admin verifies payment.
7. Admin adds order in Order Dashboard.
8. Admin delivers PDF/report/consultation.
9. Admin updates delivery status.
10. Admin sends membership upsell.

1. Open Admin Dashboard.
2. Check new WhatsApp messages.
3. Add new leads in CRM.
4. Verify payments.
5. Update orders.
6. Update revenue.
7. Send funnel follow-up messages.
8. Share store link daily.
